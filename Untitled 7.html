this page contains information on Hypertext transfer protocal and Domain Name System

HTTP (HyperText Trasfer Protocal) is the launge in which computers communicate and send information to one anthor. It is made up of GET requstes, basicliy, one computer asking anothor for information; GET{webpage name} for example


The computer that has the information requseted then sends all the code and data from the website requseted.


the data for all the websites and other things a computer could ask for is stored in a DNS (Domain Name System) server


each DNS server is resposible for a diffrent branch of the web (.net, .com, .org, .gov, .io, etc) thus ensuring that no server suffers too many requests
